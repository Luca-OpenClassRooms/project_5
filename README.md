# Installation

## Clone the repository

```
git clone https://github.com/Luca-OpenClassRooms/project_5.git
```

## Install the dependencies

```
composer install
```

## Configure app 

```
cp .env.example .env
```

## Start app 

Start the server on http://127.0.0.1:8000
```
php serve.php
```
